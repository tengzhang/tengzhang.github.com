CentOS下安装Redmine

Redmine的安装主要分为两部分：

  1. Ruby+Rails的安装
  2. Redmine的安装
    先说一下系统环境，我安装的系统是CentOS 5，选择的Redmine版本是2.5.2，Ruby的版本是1.9.3，Rails的版本是3.2.19。
    下面从以上两部分进行说明。

Ruby+Rails的安装

Ruby的安装有3中方式：

  1. 直接yum install ruby
  2. 源码编译安装
  3. 采用RVM安装
    以上3种方式我都尝试了，最后采用了RVM安装这种方式，这种方式也是最简单的。
